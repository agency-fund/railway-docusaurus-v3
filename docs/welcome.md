# Welcome

This guide will help you learn about how to become a **learning organization** by operationalizing the [AI Evaluations Framework](https://eval.playbook.org.ai/). It focuses on tech-enabled solutions but the lessons can be applied to non-tech solutions as well.

In particular, it will help you with:

1. Defining your [theory of change](theory-of-change.mdx)

2. [Building the solution](building-the-solution.mdx) (or improving an existing one)

3. Continuing to learn as you [pilot and roll out](pilots-and-rollout.mdx) the solution

4. [Evaluating its impact](evaluating-impact.mdx)

All tech-enabled solutions are built on top of a [data platform](data-platform.mdx), and could use [AI](ai.mdx) to unlock new opportunities and scale.
